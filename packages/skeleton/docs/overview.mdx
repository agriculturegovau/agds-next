## SkeletonHeading

The `SkeletonHeading` component can be used as a placeholder for any [Heading component](/packages/content/heading).

The height of the component is determined by the `type` property.

The width of the element can be customised using the `width` property.

```jsx live
<Stack gap={1.5}>
	<SkeletonHeading type="h1" />
	<SkeletonHeading type="h2" />
	<SkeletonHeading type="h3" />
	<SkeletonHeading type="h4" />
	<SkeletonHeading type="h5" />
	<SkeletonHeading type="h6" />
</Stack>
```

## SkeletonText

The `SkeletonHeading` component can be used as a placeholder for any [Text component](/packages/foundations/text).

The height of the component can be controlled using using the `fontSize` and `lineHeight` properties.

The width of the element can be customised using the `width` property.

```jsx live
<Stack gap={1.5}>
	<SkeletonText fontSize="xxxl" lineHeight="default" />
	<SkeletonText fontSize="xxl" lineHeight="default" />
	<SkeletonText fontSize="xl" lineHeight="default" />
	<SkeletonText fontSize="lg" lineHeight="default" />
	<SkeletonText fontSize="md" lineHeight="default" />
	<SkeletonText fontSize="sm" lineHeight="default" />
	<SkeletonText fontSize="xs" lineHeight="default" />
</Stack>
```

## SkeletonBox

The `SkeletonBox` component allows you to create custom shapes by providing full control over both the width and height.

```jsx live
<Stack gap={1.5}>
	<SkeletonBox width="150px" height="150px" />
	<SkeletonBox width="50%" height="50px" />
</Stack>
```
