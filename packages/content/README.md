---
title: Content
description: A set of simple layout containers for content.
group: Layout
---

### PageContent

```jsx
<ExampleApp>
	<ExampleAppHeader />
	<PageContent as="main">
		<ExampleAppContent />
	</PageContent>
	<ExampleAppFooter />
</ExampleApp>
```

### SectionContent

```jsx
<SectionContent>
	<p>Hello</p>
</SectionContent>
```

### Content

```jsx
<Content>
	<p>Blah</p>
</Content>
```
