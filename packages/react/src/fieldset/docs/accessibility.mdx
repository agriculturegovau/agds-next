Here are ways to make a Fieldset more accessible and inclusive for users of any ability or environment.

For general help with accessibility, go to the [Web Content Accessibility Guidelines 2.1](https://www.w3.org/TR/WCAG21/).

## Always provide a legend

A legend is required in order to pass [Success Criteria 1.3.1: Info and Relationships (Level A)](https://www.w3.org/WAI/WCAG21/Understanding/info-and-relationships) by using [Technique H71: Providing a description for groups of form controls using fieldset and legend elements](https://www.w3.org/WAI/WCAG21/Techniques/html/H71).
