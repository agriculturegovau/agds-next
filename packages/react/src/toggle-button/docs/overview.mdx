Toggle buttons should be used to allow a user to alternate an interaction between “on” and “off” states.

Its label should update and describe what will happen after it is pressed in its current state.

```jsx live
() => {
	const [isPressed, setPressed] = React.useState(false);

	return (
		<ToggleButton
			label={isPressed ? 'Unflag message' : 'Flag message'}
			onClick={setPressed}
			pressed={isPressed}
		/>
	);
};
```

## Available icons

To prevent inconsistent implementations, we limit the icons that can be used to current, known use-cases.

```jsx live
() => {
	return (
		<Stack alignItems="flex-start" gap={1}>
			<Flex gap={1}>
				<ToggleButton
					iconType="flag"
					onClick={console.log}
					label="flag (default)"
					pressed={false}
				/>
				<ToggleButton
					iconType="flag"
					onClick={console.log}
					label="flag (pressed)"
					pressed={true}
				/>
			</Flex>

			<Flex gap={1}>
				<ToggleButton
					iconType="star"
					onClick={console.log}
					label="star (default)"
					pressed={false}
				/>
				<ToggleButton
					iconType="star"
					onClick={console.log}
					label="star (pressed)"
					pressed={true}
				/>
			</Flex>
		</Stack>
	);
};
```

## Icon only

Some space-limited implementations may require an icon-only Toggle button. While the `label` is required for accessibility purposes, it can be visually hidden with the `hiddenLabel` prop.

```jsx live
() => {
	const [flagIsPressed, setFlagIsPressed] = React.useState(false);
	const [starIsPressed, setStarIsPressed] = React.useState(false);

	return (
		<Stack alignItems="flex-start" gap={1}>
			<ToggleButton
				iconType="flag"
				label={flagIsPressed ? 'Unflag message' : 'Flag message'}
				onClick={setFlagIsPressed}
				pressed={flagIsPressed}
				hiddenLabel
			/>
			<ToggleButton
				iconType="star"
				label={starIsPressed ? 'Remove from dashboard' : 'Add to dashboard'}
				onClick={setStarIsPressed}
				pressed={starIsPressed}
				hiddenLabel
			/>
		</Stack>
	);
};
```

## Size

Size is a property that allows you to adjust the visual weight of a button.

The medium button should be used in most cases to provide a larger icon.

```jsx live
() => {
	const [mediumIsPressed, setMediumIsPressed] = React.useState(false);
	const [smallIsPressed, setSmallIsPressed] = React.useState(false);

	return (
		<Stack alignItems="flex-start" gap={1}>
			<ToggleButton
				label="Unflag message"
				pressed={mediumIsPressed}
				onClick={setMediumIsPressed}
				iconType="flag"
				size="md"
			/>
			<ToggleButton
				label="Flag message"
				pressed={smallIsPressed}
				onClick={setSmallIsPressed}
				iconType="flag"
				size="sm"
			/>
		</Stack>
	);
};
```
