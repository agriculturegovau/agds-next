This component implements the accessibility features specified in the [Menu Button Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/menu-button/) by the [ARIA Authoring Practices Guide (APG)](https://www.w3.org/WAI/ARIA/apg/).

## ARIA roles

- The `DropdownMenuPanel` componen uses the [menu role](https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/Roles/menu_role)
- The `DropdownMenuGroup` component use the [group role](https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/Roles/group_role)
- The `DropdownMenuItem` and `DropdownMenuItemLink` components use the [menuitem role](https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/Roles/menuitem_role)
- The `DropdownMenuItemRadio` component use the [menuitemradio role](https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/Roles/menuitemradio_role)
