Filter sidebar is used in search pages to contain filtering options for a list of Cards.

```jsx live
<FilterSidebar numberOfActiveFilters={3}>
	<TextInput label="Example filtering input 1" value="Example" />
	<TextInput label="Example filtering input 2" value="Example" />
	<TextInput label="Example filtering input 3" value="Example" />
</FilterSidebar>
```

The FilterSidebar collapses on mobile devices, and displays a number of active filter options in the header.

[Visit the example page in Storybook](/storybook/index.html?path=/story/layout-filtersidebar--full) to see it used in context.
