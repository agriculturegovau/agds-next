Filter sidebar is used in search results pages to contain filtering options for a list of Cards.

The FilterSidebar collapses on mobile devices, and displays a number of active filter options in the header.

[Visit the example page in Storybook](/storybook/index.html?path=/story/layout-filtersidebar--full) to see it used in context.

<DoHeading />

- Read the [Filter a list of cards pattern](/patterns/filtering-a-dataset#filtering-a-list-of-cards) to understand how and
  when to use this component
- Use the `numberOfActiveFilters` prop to display the number of active filters in the header on mobile devices
- Add a button to 'clear filters'.

<DontHeading />

- Use outside of the context of filtering items.
