Avatars are used to show a thumbnail representation of an individual.

```jsx live
<Avatar name="Anthony Albanese" />
```

## Sizes

```jsx live
<Flex alignItems="center" gap={0.5}>
	<Avatar name="Anthony Albanese" size="sm" />
	<Avatar name="Scott Morrison" size="md" />
	<Avatar name="Malcolm Turnbull" size="lg" />
	<Avatar name="Tony Abbott" size="xl" />
	<Avatar name="Kevin Rudd" size="xxl" />
	<Avatar name="Julia Gillard" size="xxxl" />
</Flex>
```

## Colors

Avatar supports two colours: `default` and `action`.

Use the `action` color if the Avatar is used within a button or link.

```jsx live
<Flex
	as="a"
	href="#"
	gap={0.5}
	alignItems="center"
	justifyContent="flex-start"
	link
>
	<Avatar name="Anthony Albanese" size="md" color="action" />
	Anthony Albanese
</Flex>
```
