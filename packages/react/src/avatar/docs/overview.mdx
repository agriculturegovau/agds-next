Avatars are used to show a thumbnail representation of an individual.

```jsx live
<Avatar name="William Mead" />
```

## Sizes

Use to `size` prop to change the size of the avatar.

```jsx live
<Flex alignItems="center" gap={0.5}>
	<Avatar name="Taj Grainger" size="sm" />
	<Avatar name="Isabel Edmund" size="md" />
	<Avatar name="Ethan Polglaze" size="lg" />
	<Avatar name="Rachel Warlow-Davies" size="xl" />
	<Avatar name="Callum Lascelles" size="xxl" />
	<Avatar name="William Mead" size="xxxl" />
</Flex>
```

## Colors

Avatar supports two colours: `neutral` and `action`.

Use the `action` color if the Avatar is used within an interactive element such as a button or link.

```jsx live
<Flex alignItems="center" gap={0.5}>
	<Avatar name="William Mead" color="neutral" />
	<Avatar name="William Mead" color="action" />
</Flex>
```
