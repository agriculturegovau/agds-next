The Box component is a primitive layout component that is heavily used to build other components. It can also be used to create custom layouts and components in your application.

```jsx live showCode
<Box>A simple Box</Box>
```

## The ‘as’ prop

By default, the Box component renders an HTML `<div>` element. You can render it with a different HTML element tag by using the `as` prop.

```jsx live showCode
<Box as="section">A simple Box rendered as a section element</Box>
```

## Styles

Apply styles like padding, border radius, background color and more by using available props. Here’s a small example:

```jsx live showCode
<Box background="shade" border padding={1} rounded>
	A Styled Box
</Box>
```

## Breaking long words

If an unbroken word is too long to fit in in a `Box`, use the `breakWords` property to allow words to break.

Note that all children will inherit this behaviour by default. To make `breakWords` the default behaviour of your site/app,
wrap the site in a `Box` with `breakWords` set to `true`.

```jsx live
<Box css={{ overflow: 'auto' }}>
	<Stack gap={1}>
		<Box maxWidth="75%" border padding={1}>
			<strong>Default behaviour</strong>{' '}
			<Text as="p">
				aVeryLongWordThatIsUnbrokenAndNeedsToBeBrokenUpUsingCssForItToRenderNicelyOnThePage
			</Text>
		</Box>
		<Box maxWidth="75%" border padding={1} breakWords={true}>
			<Stack gap={1}>
				<div>
					<strong>
						<code>breakWords</code> = true
					</strong>{' '}
					<Text as="p">
						aVeryLongWordThatIsUnbrokenAndNeedsToBeBrokenUpUsingCssForItToRenderNicelyOnThePage
					</Text>
				</div>
				<Box border padding={1}>
					<strong>
						Child <code>breakWords</code> value is inherited from parent
					</strong>{' '}
					<Text as="p">
						aVeryLongWordThatIsUnbrokenAndNeedsToBeBrokenUpUsingCssForItToRenderNicelyOnThePage
					</Text>
				</Box>
				<Box border padding={1} breakWords={false}>
					<strong>
						Child <code>breakWords</code> = false (explicit)
					</strong>{' '}
					<Text as="p">
						aVeryLongWordThatIsUnbrokenAndNeedsToBeBrokenUpUsingCssForItToRenderNicelyOnThePage
					</Text>
				</Box>
			</Stack>
		</Box>
	</Stack>
</Box>
```
