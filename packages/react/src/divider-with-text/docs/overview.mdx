The [Divider] component with label support.

<DoHeading />

- use to create visual space and separation between blocks of content,
- provide a short text label with one or few words,
- to denote sections between optional inputs.

<DontHeading />

- use this component in substitution for a H1 element,
- use this component without a text, please refer to the [Divider component](components/divider)
- change the colour of divider or text.

## Example

```jsx live
<Stack gap={2}>
	<ControlGroup
		label="Business structure"
		hint="When you make a selection, you will be asked for your Australian Business Number (ABN) or Australian Company Number (ACN)."
		block
		required
	>
		<Radio checked={true}>Sol trader</Radio>
		<Radio checked={false}>Business</Radio>
		<Radio checked={false}>Company</Radio>
	</ControlGroup>
	<DividerWithText label="Business" labelAlign="left" />
	<TextInput label="Australian Business Number (ABN)" required />
</Stack>
```

## Label alignment

The label location and text wrap alignment can be assigned by using the `labelAlignment` prop. The values `center` or `left` will align the label towards the respective locations.

```jsx live
<Stack gap={2}>
	<DividerWithText label="center" />
	<DividerWithText label="left" labelAlign="left" />
</Stack>
```

## Font sizes

The label size is determined by the element used in the `as` props as the default.

```jsx live
<Stack gap={2}>
	<DividerWithText label="H2" as="h2" />
	<DividerWithText label="H3" as="h3" />
	<DividerWithText label="H4" as="h4" />
	<DividerWithText label="H5" as="h5" />
	<DividerWithText label="H6" as="h6" />
	<DividerWithText label="p" as="p" />
	<DividerWithText label="span" as="span" />
</Stack>
```

The `fontSize` prop can be used in combination with the `as` prop to control the heading element and size.

```jsx live
<Stack gap={2}>
	<DividerWithText
		label="Paragraph with XL font size"
		as="p"
		fontSize="xl"
		labelAlign="left"
	/>
	<DividerWithText
		label="H2 with SM font size"
		as="h2"
		fontSize="sm"
		labelAlign="left"
	/>
</Stack>
```
