Call to action

Call to action links are interactive UI cues that direct users to other pages.

View in FigmaView in StorybookView in Github

Usage

Call to action components can be imported via the call-to-action entrypoint in the @ag.ds-next/react package.

For example:

import { ... } from '@ag.ds-next/react/call-to-action';
