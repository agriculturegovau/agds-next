## Selectable tables

Selectable [Tables](/components/table) allow users to select one or more rows in a table simultaneously.

[View an example in Storybook](/storybook/index.html?path=/story/content-table--selectable-basic)

### Individual row select

To select an individual row, users can use the [Checkbox](/components/checkbox) in the first column ("Select") to toggle the selection of an individual row. Checkboxes in each row have only two states: checked and unchecked.

```jsx live
() => {
	const [isRowSelected, setIsRowSelected] = useState(false);
	return (
		<Table>
			<TableHead>
				<TableRow>
					<TableHeader scope="col">Select</TableHeader>
					<TableHeader scope="col">Example Column</TableHeader>
					<TableHeader scope="col">Example Column</TableHeader>
				</TableRow>
			</TableHead>
			<TableBody>
				<TableRow selected={isRowSelected}>
					<TableCell>
						<Checkbox
							size="sm"
							checked={isRowSelected}
							onChange={() => setIsRowSelected((x) => !x)}
						>
							<VisuallyHidden>Select row</VisuallyHidden>
						</Checkbox>
					</TableCell>
					<TableCell>Example Row</TableCell>
					<TableCell>Example Row</TableCell>
				</TableRow>
			</TableBody>
		</Table>
	);
};
```

### Selecting all rows

Users can toggle the selection of all rows at once by toggling the [Checkbox](/components/checkbox) above the table. Unlike the Checkboxes in each row, the "Select all rows" checkbox above the table has three states: checked, unchecked, and indeterminate.

When some, but not all, rows are selected, the "Select all rows" checkbox should enter an indeterminate state. This visual indicator signals to users that not all rows are selected and allows them to toggle all rows on or off with a single click.

The "Select all rows" checkbox should always be placed just above the Table element, with a border to divide the two elements. Don't use `TableCaption` component as it will sit in between the Checkbox and Table. Instead, using a [Heading](/components/heading) component and connext the heading and table via using using `aria-labelledby`. For more information, please refer to the [Labels and headings](/components/table#labels-and-headings) section of the Table documentation.

```jsx live
() => {
	const [selectedRows, setSelectedRows] = useState([]);

	function toggleRowSelection(rowIdx) {
		if (selectedRows.includes(rowIdx)) {
			setSelectedRows((r) => r.filter((s) => s !== rowIdx));
		} else {
			setSelectedRows((r) => [...r, rowIdx]);
		}
	}

	const anyRowsChecked = selectedRows.length > 0;
	const allRowsChecked = selectedRows.length === 3;

	function toggleAllRows() {
		if (anyRowsChecked) {
			setSelectedRows([]);
		} else {
			setSelectedRows([0, 1, 2]);
		}
	}

	return (
		<Stack>
			<Box paddingBottom={0.75} paddingLeft={0.75} borderBottom>
				<Checkbox
					size="sm"
					checked={allRowsChecked}
					indeterminate={anyRowsChecked && !allRowsChecked}
					onChange={() => toggleAllRows()}
				>
					Select all rows
				</Checkbox>
			</Box>
			<Table>
				<TableHead>
					<TableRow>
						<TableHeader scope="col">Select</TableHeader>
						<TableHeader scope="col">Example Column</TableHeader>
						<TableHeader scope="col">Example Column</TableHeader>
					</TableRow>
				</TableHead>
				<TableBody>
					{Array.from(new Array(3).keys()).map((idx) => {
						const isRowSelected = selectedRows.includes(idx);
						return (
							<TableRow selected={isRowSelected}>
								<TableCell>
									<Checkbox
										size="sm"
										checked={isRowSelected}
										onChange={() => toggleRowSelection(idx)}
									>
										<VisuallyHidden>Select</VisuallyHidden>
									</Checkbox>
								</TableCell>
								<TableCell>Example Row {idx + 1}</TableCell>
								<TableCell>Example Row</TableCell>
							</TableRow>
						);
					})}
				</TableBody>
			</Table>
		</Stack>
	);
};
```

## Batch actions

When one or more rows in a table are selected, users can perform batch actions on the selected rows. Batch actions can increase user efficiency compared to the effort of repeatedly performing the same action across multiple table rows.

[View an example in Storybook](/storybook/index.html?path=/story/content-table--selectable-with-batch-actions)

Once a row from the table has been selected, the `TableBatchActionsBar` component appears at the bottom of the table, presenting a set of possible actions to apply to all selected items.

Only [Buttons](/components/button) should be placed inside of the `TableBatchActionsBar` component. Do not place form components inside the `TableBatchActionsBar` component, as they can introduce complexity related to validation and submission.

Action buttons should trigger [Drawers](/components/drawer) when pressed, which should contain the form related for each button. Each action button should be applicable for every row in the Table. If certain actions can not be performed on every selectable row, do not include this action in the `TableBatchActionsBar`.

```jsx live
() => {
	const [selectedRows, setSelectedRows] = useState([]);

	function toggleRowSelection(rowIdx) {
		if (selectedRows.includes(rowIdx)) {
			setSelectedRows((r) => r.filter((s) => s !== rowIdx));
		} else {
			setSelectedRows((r) => [...r, rowIdx]);
		}
	}

	const anyRowsChecked = selectedRows.length > 0;
	const allRowsChecked = selectedRows.length === 3;

	function toggleAllRows() {
		if (anyRowsChecked) {
			setSelectedRows([]);
		} else {
			setSelectedRows([0, 1, 2]);
		}
	}

	return (
		<Stack gap={0.5}>
			<Stack>
				<Box paddingBottom={0.75} paddingLeft={0.75} borderBottom>
					<Checkbox
						size="sm"
						checked={allRowsChecked}
						indeterminate={anyRowsChecked && !allRowsChecked}
						onChange={() => toggleAllRows()}
					>
						Select all rows
					</Checkbox>
				</Box>
				<Table>
					<TableHead>
						<TableRow>
							<TableHeader scope="col">Select</TableHeader>
							<TableHeader scope="col">Example Column</TableHeader>
							<TableHeader scope="col">Example Column</TableHeader>
						</TableRow>
					</TableHead>
					<TableBody>
						{Array.from(new Array(3).keys()).map((idx) => {
							const isRowSelected = selectedRows.includes(idx);
							return (
								<TableRow selected={isRowSelected}>
									<TableCell>
										<Checkbox
											size="sm"
											checked={isRowSelected}
											onChange={() => toggleRowSelection(idx)}
										>
											<VisuallyHidden>Select</VisuallyHidden>
										</Checkbox>
									</TableCell>
									<TableCell>Example Row {idx + 1}</TableCell>
									<TableCell>Example Row</TableCell>
								</TableRow>
							);
						})}
					</TableBody>
				</Table>
			</Stack>
			{selectedRows.length > 0 ? (
				<TableBatchActionsBar>
					<TableBatchActionsTitle>
						Apply action to {selectedRows.length} items
					</TableBatchActionsTitle>
					<ButtonGroup>
						<Button variant="secondary" size="sm">
							Example action
						</Button>
						<Button variant="tertiary" size="sm">
							Delete
						</Button>
					</ButtonGroup>
				</TableBatchActionsBar>
			) : null}
		</Stack>
	);
};
```
